# calculator
Aplicação de calculadora simples que realiza operações matemáticas básicas como adição, subtração, multiplicação e divisão. Projeto criado para praticar conceitos fundamentais de programação, lógica matemática e desenvolvimento de interfaces.

Listar (opcional, só para conferir):
ls ~/Desktop

Entrar na pasta base:
cd ~/Desktop/odin_project

Clonar o repositório:
git clone https://github.com/seu-usuario/calculator.git

Entrar na nova pasta criada:
cd calculator

Abrir no VS Code:
code .
